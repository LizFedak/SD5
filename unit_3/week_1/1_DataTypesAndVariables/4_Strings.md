### Strings
* A string can be defined with either single or double quotes:
  
```javascript
var firstName = "Steven";
var lastName = 'Gregory';
```
  
* String comparison is possible using only the `==` comparator:
  
```javascript
var x = 'hello';
x == 'hello' // true
x == 'squirrel' // false
```
