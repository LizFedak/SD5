#### Objective
In Quad 2 we are going to start creating a deploying java web applications. Using Servlets and JSP's, Spring MVC, SQL, JDBC, and JPA, we can create a web application with a functional back end. In addition, we will use an Amazon EC2 server instance to deploy our apps to a publicly accessible location.
